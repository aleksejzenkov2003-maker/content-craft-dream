

## Вынос промтов в настройки — ВЫПОЛНЕНО ✅

### Что сделано

1. **Добавлены 3 новых типа промтов**: `atmosphere`, `scene`, `post_text` в PromptForm и PromptLibrary с соответствующими переменными
2. **Засеяны дефолтные промты** в таблицу `prompts` с текущими захардкоженными текстами
3. **Edge Functions обновлены** — все 3 функции читают активный промт из БД:
   - `generate-cover` → тип `atmosphere`
   - `generate-scene` → тип `scene`  
   - `generate-post-text` → тип `post_text` (fallback если у канала нет своего)
4. **VideoSidePanel** — добавлена сворачиваемая секция с textarea для промта атмосферы перед генерацией
5. **SceneSidePanel** — автоматическое предзаполнение промта из настроек если поле пустое
