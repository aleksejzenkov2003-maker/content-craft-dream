

## Добавить чекбоксы и массовую генерацию сцен

### Изменения в `src/components/scenes/ScenesMatrix.tsx`

1. **Состояние выбранных** — добавить `selectedPairs: Set<string>` (ключ `playlistId-advisorId`)

2. **Чекбокс в каждой строке плейлиста** — перед названием плейлиста добавить `<Checkbox>`, клик по которому добавляет/убирает пару из `selectedPairs`. Клик по чекбоксу останавливает propagation (чтобы не открывался side panel).

3. **Кнопка массовой генерации** — при `selectedPairs.size > 0` показывать `BulkActionsBar` снизу (или кнопку в шапке) с текстом «Сгенерировать N сцен». По клику — последовательно вызывать `handleGenerateScene` для каждой выбранной пары, очищать selection после завершения.

4. **Чекбокс «выбрать всех» у адвайзера** — в заголовке каждого адвайзера добавить чекбокс, который выделяет/снимает все плейлисты этого адвайзера.

### Логика массовой генерации

Сцены генерируются **последовательно** (по одной), чтобы не перегружать API. Каждая пара добавляется в `generatingScenes` при старте и убирается при завершении. После окончания всей очереди — `refetch()` и тост с итогом.
